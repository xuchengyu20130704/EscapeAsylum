# 🏥 逃离精神病院

一个基于Godot 4.x的沉浸式剧情游戏，玩家通过选择影响故事走向。

## 🎮 游戏简介

你被关进了北山精神病院，但你坚信自己没病。今晚，你终于找到了逃跑的机会...
你的选择将决定故事的走向：成为英雄？沦为罪犯？还是隐姓埋名？

## ✨ 特性

- ✅ 多分支剧情系统
- ✅ 道德值/理智值影响结局
- ✅ 3条主要故事线（正派/反派/回归）
- ✅ 存档/读档功能
- ✅ 状态显示系统
- ✅ 模块化设计，易于扩展

## 🎯 游戏玩法

1. 阅读剧情
2. 做出选择
3. 影响状态
4. 走向不同结局

## 📁 项目结构
逃离精神病院/
├── assets/ # 资源文件（图片、音乐、音效）
├── modules/ # 剧情模块
│ ├── 基础模块.gd # 主线剧情
│ ├── 反派线.gd # 邪恶路线
│ ├── 正派线.gd # 正义路线
│ └── 回归线.gd # 普通生活
├── scenes/ # 场景文件
├── scripts/ # 脚本文件
├── saves/ # 存档文件
└── project.godot # 项目配置

## 🚀 如何运行

1. 安装 [Godot 4.x](https://godotengine.org/download)
2. 克隆本仓库
3. 用Godot打开 `project.godot`
4. 点击运行
